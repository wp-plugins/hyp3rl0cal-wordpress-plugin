== Description ==

Plugin for deploying a local business directory, using the CityGrid API

== Installation ==

1. Upload CityGrid Hyp3rL0cal Wordpress Pluging to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust settings

== Remove plugin ==

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress

It's best to use the build in delete function of wordpress. That way all the stored data will be removed and no orphaned data will stay.
